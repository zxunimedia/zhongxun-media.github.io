# 💼 SOHO專案管理系統

一個專為SOHO族設計的專案管理與財務追蹤應用程式,具有動態液體玻璃效果的精美UI設計。

## ✨ 核心功能

### 📅 智能行事曆
- 週曆視圖,自動顯示週一至週日
- 自動更新年份與日期
- 台灣節日標註功能
- 任務點標示
- 今日高亮顯示

### 💰 財務管理
- 自動計算月度收支
- 收入/支出分類記錄
- 即時淨利計算
- 專案別財務追蹤
- 金額格式化顯示

### 📋 專案管理
- 專案建立與追蹤
- 客戶資訊管理
- 簽約金額記錄
- 任務進度追蹤
- 專案狀態統計

### ⏰ 智能提醒
- 自訂提醒時間
- 重複任務設定
  - 每天重複
  - 每週重複
  - 自訂週期(例:每週二、四)
- 瀏覽器通知提醒
- 今日待辦清單

### 🎯 重複任務模板
特別適合「廣告代po 一個月8則,固定週二週四」等固定週期任務:
- 自動生成未來3個月的任務實例
- 自訂每週哪幾天執行
- 單獨完成每次任務
- 自動任務計數

## 📱 使用方式

### 安裝
1. 下載所有檔案到同一資料夾
2. 用瀏覽器開啟 `index.html`
3. 建議使用手機瀏覽器或開啟開發者工具的手機模式

### 基本操作

#### 新增專案
1. 點擊右下角「+」浮動按鈕
2. 選擇「📁 新增專案」
3. 填寫專案資訊
4. 點擊「建立專案」

#### 新增任務
1. 點擊「+ 新增專案」或浮動按鈕
2. 選擇「✓ 新增任務」
3. 填寫任務資訊
4. 設定重複週期(選填)
   - 選擇「自訂週期」
   - 勾選要執行的星期幾
5. 完成建立

#### 記帳
1. 點擊浮動按鈕
2. 選擇「💰 記帳」
3. 選擇收入/支出
4. 填寫金額與項目
5. 可選擇所屬專案

#### 完成任務
- 在「今日提醒」區塊點擊任務右側的圓圈

#### 查看詳情
- 點擊專案卡片查看專案詳情
- 點擊日期查看當日任務

## 🎯 特殊功能說明

### 重複任務範例

**情境：廣告代po 一個月8則,固定週二、週四**

1. 新增任務:「廣告代po」
2. 設定第一次執行日期
3. 重複設定選擇:「自訂週期」
4. 勾選:「二」和「四」
5. 系統會自動生成未來3個月每週二、四的任務

### 節日設定

目前內建台灣主要節日:
- 元旦 (1/1)
- 農曆春節 (2/10)
- 和平紀念日 (2/28)
- 兒童節 (4/4)
- 清明節 (4/5)
- 勞動節 (5/1)
- 端午節 (6/10)
- 中秋節 (9/17)
- 國慶日 (10/10)

可在 `app.js` 的 `initHolidays()` 方法中自訂更多節日。

## 💡 使用建議

1. **每天早上** - 查看「今日提醒」,確認待辦事項
2. **接到新案子** - 立即建立專案,記錄簽約金
3. **收到款項** - 使用記帳功能記錄收入
4. **有支出時** - 記錄支出項目與金額
5. **月底時** - 查看「本月收支總覽」,了解財務狀況

## 🔔 瀏覽器通知

首次使用時,請允許瀏覽器通知權限,以便在任務時間到達時收到提醒。

**開啟通知權限:**
1. 點擊網址列的鎖頭圖示
2. 找到「通知」設定
3. 選擇「允許」

## 🚀 部署到 GitHub

### 步驟

1. **建立 GitHub 倉庫**
   ```bash
   # 在專案資料夾中
   git init
   git add .
   git commit -m "Initial commit: SOHO專案管理系統"
   ```

2. **連接遠端倉庫**
   ```bash
   git remote add origin https://github.com/你的帳號/soho-project-manager.git
   git branch -M main
   git push -u origin main
   ```

3. **開啟 GitHub Pages**
   - 進入倉庫設定 (Settings)
   - 找到 Pages 選項
   - Source 選擇 `main` 分支
   - 儲存後等待部署完成

4. **訪問網址**
   - `https://你的帳號.github.io/soho-project-manager/`

## 🔧 客製化建議

### 修改主題顏色
在 `styles.css` 頂部的 `:root` 區塊修改顏色變數:

```css
:root {
    --primary-dark: #0a0e27;        /* 主背景色 */
    --accent-gold: #f4c542;         /* 強調色 */
    --accent-purple: #8b5cf6;       /* 輔助色 */
    /* ... */
}
```

### 新增專案類型
在 `app.js` 的 `getProjectTypeName()` 方法中新增:

```javascript
const types = {
    'social-media': '社群媒體',
    'design': '設計',
    'your-type': '你的類型',  // 新增這裡
    // ...
};
```

同時在 `index.html` 的專案表單中新增選項。

### 調整自動生成任務期間
在 `app.js` 的 `generateRecurringTasks()` 方法中修改:

```javascript
endDate.setMonth(endDate.getMonth() + 3); // 改成你要的月數
```
